# LimpiadorFS ESP8266/ESP32-CAM

Este proyecto permite limpiar y gestionar el sistema de archivos (SPIFFS/LittleFS) en dispositivos **ESP8266** y **ESP32-CAM**.  
Su objetivo es facilitar la depuración y mantenimiento de proyectos embebidos eliminando archivos innecesarios y mostrando el estado del sistema.
